# Red Shield: Installation & Launch Guide

Welcome to the Red Shield network. In this guide, we will walk you through the simple process of installing your SIMBA Prime Node on a Windows machine, and launching your Quantum Native Wallet on Android.

---

## Part 1: Installing the Red Shield Node (MSI)

**1. Running the Installer**
Locate the `RedShieldNode_v11.msi` installer file you downloaded from the main README. Double-click the file to begin. You'll be greeted by the Welcome screen. Click 'Next' to continue.

*(See reference: `../RED SHIELD - SIMBA PRIME - BetaNet_NODE - v1.1.0/Installation_Screens/Screen1.png`)*

**2. End-User License Agreement**
Take a moment to review the License Agreement. Once you're ready, check the box to accept the terms, and then click 'Next'.

*(See reference: `../RED SHIELD - SIMBA PRIME - BetaNet_NODE - v1.1.0/Installation_Screens/Screen2.png`)*

**3. Choosing the Install Location**
By default, the Red Shield Node will install to your Program Files directory. If you want to install it on a different drive, click 'Change' and select your preferred folder. Otherwise, leave it as is and click 'Next'.

*(See reference: `../RED SHIELD - SIMBA PRIME - BetaNet_NODE - v1.1.0/Installation_Screens/Screen3.png`)*

**4. Beginning the Installation**
Click 'Install' to begin writing the files to your computer. Windows may prompt you with a User Account Control (UAC) pop-up asking if you want to allow the app to make changes. Click 'Yes' to authorize the installation.

*(See references: `../RED SHIELD - SIMBA PRIME - BetaNet_NODE - v1.1.0/Installation_Screens/Screen4.png` and `Screen5.png`)*

**5. Installation Complete**
And that's it! Your Red Shield Node is now successfully installed. Click 'Finish' to exit the setup wizard. Your node will now run quietly in the background, securing the network and earning Progressive Trust Score (PTS) rewards.

*(See reference: `../RED SHIELD - SIMBA PRIME - BetaNet_NODE - v1.1.0/Installation_Screens/Screen6.png`)*
